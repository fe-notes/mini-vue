## Date

### Date() 和 new Date()

- `Date()` 返回的是当前时间的字符串
- `new Date()` 返回的是当前时间的对象

```js
+new Date(); // <- 1677910166753
```

### Date.getMonth()

根据本地时间，返回一个指定的日期对象的月份，为基于 0 的值（0 表示一年中的第一月）。

### Date.getDate()

根据本地时间，返回一个指定的日期对象的月份中的一天（1 到 31）。

### Date.getFullYear()

根据本地时间，返回一个指定的日期对象的年份。
